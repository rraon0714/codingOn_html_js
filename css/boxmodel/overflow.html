overflow 속성 visible default

요소의 box에 내용이 더 길 떄 어떻게 보일지 선택하는 속성 visible: 자식요소가 부모 요소의 범위를 넘어가더라도 보여준다. scroll: 스크롤러가 추가되어 스크롤 할 수 있다. hidden: 부모 요소의 범위를 넘어가는 자식 요소의 부분은 보이지 않는다. autoL 내용이 넘치지 않으면 그대로 표시되고,내용이 넘치면 스크롤바를 표시한다.
